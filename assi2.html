String Functions

1).Length : Length method return the length of string

2).toLowerCase : Return All string in lower case

3).toUpperCase : Return all string in Upper case

4).substring : Returns the substring at the specified location within a String object.

5).replace :replace specific value from another value in string object.

6).indexOf :returns the index of (the position of) the first occurrence of a specified text in a string

7).lastindexOf :second parameter as the starting position for the search.
